# Transaction Record Analysis

In this project, I analyzed daily bank deposit transactions using Python. The objective was to apply fundamental Python programming concepts to organize transaction records, perform basic financial calculations, and update transaction data using Python lists and built-in functions.

#### The dataset consists of five daily deposit transactions:

5000, 12000, 7500, 3000, 9000

Using this dataset, I performed a series of operations to calculate important financial metrics, identify the highest and lowest deposits, update the transaction records with a new deposit, and remove the smallest transaction. This project demonstrates how Python simplifies basic financial data analysis and record management.

The objectives of this project were to:

- Store bank deposit amounts in a Python list.
- Calculate the total amount deposited.
- Identify the highest deposit transaction.
- Identify the lowest deposit transaction.
- Add a new deposit transaction to the dataset.
- Remove the smallest deposit from the dataset.
- Display the final updated transaction records.

#### This project strengthened my understanding of:

- Creating and managing Python lists.
- Performing basic financial calculations.
- Using built-in functions to analyze numerical data.
- Updating datasets by adding and removing records.
- Writing clean and readable Python code.
- Applying Python to solve simple financial data analysis problems.

#### By completing this project, I learned how to:

- Organize financial transaction data using Python lists.
- Calculate summary statistics from numerical datasets.
- Identify the highest and lowest values within a dataset.
- Update transaction records dynamically.
- Apply Python fundamentals to practical financial analysis scenarios.

This project introduces the fundamentals of financial data analysis using Python. Although the dataset is small, the workflow reflects the basic processes used in transaction management and financial reporting.
